CSP solver

An implementation of a Constraint Satisfaction Problem solver. CSP problems are defined as a set of variables, each with an associated domain, and a set of contstraint to be satisfied. The goal is to find an assignment of each variable, such that every constraint is satisfied. The problem is solved as a search in the space of assignment and is of course a NP problem. Thanks to smart heuristics, a CSP problem can often be solved very efficiently. My implementation features the following classic heuristics: forward propagation, generalized arc concistency, minimum remaining values and max degree.

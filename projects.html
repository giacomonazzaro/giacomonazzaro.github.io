CSP solver

A constraint satisfaction problem is defined as a set of variables, each with an associated domain, and a set of contstraints. The goal is to find an assignment for each variable, such that every constraint is satisfied. The problem is solved as a search in the vast space of possible assignments. My solver features forward propagation, GAC3, MRV and max degree.

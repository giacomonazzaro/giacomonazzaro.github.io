CSP solver

Constraint Satisfaction Problems are defined as a set of variables, each with an associated domain, and a set of contstraint to be satisfied. The goal is to find an assignment for each variable, such that every constraint is satisfied. The problem is solved as a search in the space of assignments and is of course a NP problem. My implementation features the following classic heuristics: forward propagation, generalized arc concistency, minimum remaining values and max degree.
